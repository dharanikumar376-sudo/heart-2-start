How The System Works

  1. Helmet Detection: The Force-Sensitive Resistor (FSR) sensor inside the helmet detects when it's being worn.
  2. Wireless Communication: The helmet's Bluetooth module sends an "ON" signal to the bike unit.
  3. Authentication: The bike's ECU verifies the signal is from an authorized helmet.
  4. Ignition Enable: If verified, the relay completes the fuel pump circuit, allowing the engine to start.
  5. Safety Feature: If the helmet is removed while riding, the system cuts power to the fuel pump, stopping the engine.
